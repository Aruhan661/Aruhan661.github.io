八皇后问题

八皇后问题是一个以国际象棋为背景的问题：如何能够在8×8的国际象棋棋盘上放置八个皇后，使得任何一个皇后都无法直接吃掉其他的皇后？为了达到此目的，任两个皇后都不能处于同一条横行、纵行或斜线上。八皇后问题可以推广为更一般的n皇后摆放问题：这时棋盘的大小变为n×n，而皇后个数也变成n。当且仅当n = 1或n ≥ 4时问题有解。

八皇后问题最早是由国际象棋棋手Max Bezzel于1848年提出。第一个解在1850年由Franz Nauck给出。

汉诺塔问题

Posted on 2023-11-07 In Program , Classic Puzzle

汉诺塔

有三根杆子A，B，C。A杆上有 N 个 (N>1) 穿孔圆盘，盘的尺寸由下到上依次变小。要求按下列规则将所有圆盘移至 C 杆：

  1. 每次只能移动一个圆盘；
  2. 大盘不能叠在小盘上面。

提示：可将圆盘临时置于 B 杆，也可将从 A 杆移出的圆盘重新移回 A 杆，但都必须遵循上述两条规则。

问：如何移？最少要移动多少次？

递归解决汉诺塔问题 C++

假设有 A、B、C 三个塔，A 塔有N块盘，目标是把这些盘全部移到 C 塔。那么先把 A 塔顶部的N-1块盘移动到 B 塔，再把 A 塔剩下的大盘移到 C，最后把 B 塔的 N-1 块盘移到 C。

主成分分析

Posted on 2023-11-01 Edited on 2023-11-07 In Machine Learning

主成分分析法 PCA

在机器学习和数据分析问题中，通常需要处理大规模多变量的数据集。这些数据集中的众多变量可能相互关联，同时可能包含冗余信息和无用的噪声。这可能会导致以下问题：

  1. 样本不足： 在高维度数据中，可能需要更多的样本才能对模型进行有效的训练。
  2. 计算复杂性的增加： 处理高维数据需要更多的计算资源。
  3. 过拟合风险： 多变量数据中的冗余信息和噪声可能使模型更容易过拟合。
  4. 解释性困难： 在高维度数据中，理解和解释模型的结果变得更加困难。

为了解决这一问题，数据降维成为一种关键技术。

数据降维是一种在机器学习和数据分析领域常用的技术，旨在减少数据集的维度，同时尽可能保留数据中重要的信息。数据降维的主要目的是降低计算复杂性、减少噪声和冗余，以便更有效地处理数据并提高模型性能。

主成分分析法就是一个常用的数据降维的方法。

主成分分析的基本思想

主成分分析 Principal components analysis（PCA）的核心思想是将原始 $n$ 维特征映射到新的 $k$ 维空间上，其中这 $k$ 维是正交的、全新的特征，也称为主成分。这个映射过程的目标是重新构造数据的特征空间，以更好地表示原始数据的重要信息。PCA的工作过程如下：

  1. 均值中心化： 首先，将数据的坐标轴中心移到数据的平均值中心，以使数据的中心为原点。这有助于处理数据的尺度问题，确保数据的均值为零。

  2. 选择主成分： PCA的核心是找到新的坐标轴，这些坐标轴是原始特征的线性组合，并且它们是正交的（彼此之间垂直）。

    1. 将坐标轴中心移到数据的中心，然后旋转坐标轴，使得数据在 $C_1$ 轴上的方差最大，即全部 $n$ 个数据个体在该方向上的投影最为分散。意味着更多的信息被保留下来。$C_1$ 成为第一主成分。

    2. $C_2$第二主成分：找一个$C_2$，使得$C_2$与$C_1$的协方差（相关系数）为 $0$，以免与$C_1$信息重叠，并且使数据在该方向的方差尽量最大。

    3. 以此类推，找到第三主成分，第四主成分……第 $p$ 个主成分。 $p$ 个随机变量可以有 $p$ 个主成分

  3. 投影数据： 在新的坐标轴上投影原始数据。这意味着将数据点映射到主成分上，得到新的数据集，其中每个数据点都由主成分构成。

  4. 降维： 通常，只保留前 $k$ 个主成分。这意味着只保留包含最大方差的主成分，忽略包含几乎没有方差的主成分。这实现了数据降维，减少了特征的维度。

通过这个过程，PCA有助于减少数据维度，同时保留尽可能多的信息，因为主成分包含了原始数据中的主要变化方向。这使得数据更易于处理、可视化和解释。

可以通过协方差矩阵的性质来求得数据集中的主成分

PCA中的协方差矩阵

协方差： 用于衡量两个随机变量间的相关程度

协方差矩阵： 协方差矩阵描述了特征之间的线性关系。对于一个包含 $n$ 个特征的数据集，协方差矩阵的元素表示 $c[i][j]$ 特征 $i$ 和特征 $j$ 之间的协方差。

协方差矩阵的特征值： 协方差矩阵的特征值是一个数值，它表示了数据在相应特征向量方向上的方差，即数据集在这个方向上的变化程度。特征值的大小决定了主成分的重要性，即它们表示了数据中的主要方向。

  • 较大的特征值对应的主成分包含了数据中更多的方差，因此对数据的解释能力更强。
  • 较小的特征值对应的主成分包含了数据中较少的方差，因此对数据的解释能力较弱。
  • 特征值为零表示主成分没有方差，也就是说在该方向上没有数据的变化。

特征值通常按降序排列，最大的特征值对应第一个主成分，第二大的特征值对应第二个主成分，依此类推。

协方差矩阵的特征向量： 协方差矩阵的特征向量是单位向量，它们描述了数据集中主成分的方向。

主成分分析算法

  1. 对原始数据进行均值中心化，即将每个特征的均值减去数据集中的均值。
  2. 计算协方差矩阵。
  3. 对协方差矩阵进行特征值分解，得到特征值和对应的特征向量。
  4. 特征值降序排序，选择前 $k$ 个主成分作为新的特征向量。
  5. 在新的坐标系中，将原始数据点投影到主成分上，得到新的数据集。
